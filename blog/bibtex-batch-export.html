How to Batch download bibtex data or how to batch export bibtex from google scholar?

Recently, I have to search and read some papers related to my research work.

At first time, I forgot to download the reference format of bibtex, when I want to site

someone’s paper, It’s boring for me to copy the title and search from scholar.google.com once and once again.

So I wrote a python script for me to download more than one paper per time. I have checked this script in my local computer, and it works well. Since I thought someone else may need this function, I changed this local script to Web version as above. If you cannot find it in this page, you can click HERE.

You just need to copy the titles to the textarea(one line per paper), and then click “Get it”, after a while, you’ll get all

the references format of bibtex. Hope this script is useful.
